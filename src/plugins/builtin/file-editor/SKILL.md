# VSSH File Editor Plugin

Use this plugin when the task is a structured file mutation rather than raw shell editing.

## Primary Commands
- `vssh ef <path> --search "old" --replace "new"`
- `vssh ef <path> --regex "pattern" --with "value" --flags "g"`
- `vssh ef <path> --insert-at <line> --content "..."`
- `vssh ef <path> --delete-line <n>`
- `vssh ef <path> --edits '[...]'`
- `vssh ef <path> --dry-run ...`

## Workflow
1. Prefer `--dry-run` first for anything non-trivial.
2. Use simple search/replace when possible.
3. Use regex or JSON edit batches only when needed.
4. Verify the resulting file afterward.

## Examples
```bash
vssh ef /etc/app.conf --search "localhost" --replace "example.com" --dry-run
vssh ef script.sh --regex "console\\.log" --with "// console.log" --flags "g"
vssh ef app.conf --insert-at 0 --content "# Generated by vssh"
```

## Safety
- Critical system files are guard-blocked.
- Keep backups enabled unless the user explicitly asks otherwise.
- Prefer narrow edits over broad regex changes.
